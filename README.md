# CGSA Website

## Centre of Gandhian Studies and Action

A modern web platform dedicated to promoting Gandhian values through education, articles, and historical insights.

### Features

* User Authentication
* Publish Articles
* Edit & Delete Articles
* Gandhian Timeline
* About Us Page
* Contact Us Page
* Dark Mode Support
* Responsive Design
* Modern UI with Tailwind CSS
* GSAP Animations

### Tech Stack

#### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router
* GSAP
* React Toastify

#### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

### Installation

Frontend

cd Frontend
npm install
npm run dev


Backend

cd Backend
npm install
npm start
